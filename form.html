This is a form example

  • Use "for" attr of "label" tag to specify what element u are labeling. This Enables additional features like,activating a input field when clicked on label
  • Note the two attributes of "input tag", "id" & "name" .
    • The id can be used to specify style for the input in the .css file.
    • The name is used in server side languages like javascript,python
    • Its good practice to half same value for both the id & name attribute like we've done here.
  • Notice the "method" attribute on the form tag.it's value can be "post" or "get". If you know about http you probably know about these.
